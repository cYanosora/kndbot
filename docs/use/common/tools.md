## 翻译
* 说明  
    会生草的翻译机你喜欢吗？  
* 指令  
    `翻译`+`[文本]` ：默认翻译为中文  
    `翻英`/`英翻`+`[文本]` ：翻译为英文  
    `翻译英文`/`英文翻译`+`[文本]`  ：翻译为英文  
    `翻日`/`日翻`+`[文本]` ：翻译为日文  
    `翻译日文`/`日文翻译`+`[文本]`  ：翻译为日文  
    `翻韩`/`韩翻`+`[文本]` ：翻译为韩文  
    `翻译韩文`/`韩文翻译`+`[文本]` ：翻译为韩文  
    `翻法`/`法翻`+`[文本]` ：翻译为法文  
    `翻译法文`/`法文翻译`+`[文本]` ：翻译为法文  
    `翻德`/`德翻`+`[文本]` ：翻译为德文  
    `翻译德文`/`德文翻译`+`[文本]` ：翻译为德文  
    `翻俄`/`俄翻`+`[文本]` ：翻译为俄文  
    `翻译俄文`/`俄文翻译`+`[文本]` ：翻译为俄文
